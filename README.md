# API Portal

To start the server: **npm start**

*OR*
**npm install pm2**

**pm2 start npm --name "HackathonAPI" -- start --watch**

Make a GET request to any of the end points.
[Postman](https://www.getpostman.com/) works well for testing.

## End point list:
- /api/v1/

- /api/v1/people
- /api/v1/expedition
- /api/v1/department
- /api/v1/hall

- ?search possible across all
